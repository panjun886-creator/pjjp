# pjjp
# 诸葛亮资料库（老哥）

这个仓库用于固定我的写作口径、风格母版、模板与项目状态。每次写作/出稿前，默认先检索本仓库再动笔。

## 目录
- 00_core/：协作协议、禁区、口径（稳定规则）
- 01_style_samples/：风格母版（你最满意的样文/样段）
- 02_templates/：模板库（栏目模板、清单、条款表达）
- 03_project_state/：项目状态（连载进度、边界、已写/待写、避免重复）

## 更新规则（很重要）
- 新增/修改规则：只写进 00_core
- 新增母版：只放 01_style_samples
- 新增模板：只放 02_templates
- 连载推进/边界变化：只改 03_project_state
